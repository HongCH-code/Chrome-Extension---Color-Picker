# Color Picker - Chrome 擴充套件

一個功能強大的網頁取色工具，支援 HEX、RGB、HSL 格式和調色板生成。

## 功能特色

- **網頁取色** - 點擊「PICK COLOR」按鈕即可從網頁上吸取任何顏色
- **即時顏色選擇器** - 使用色相、飽和度、亮度滑桿選擇顏色
- **多種顏色格式** - 支援 HEX、RGB、HSL 格式
- **一鍵複製** - 快速複製任何格式的顏色代碼
- **顏色歷史記錄** - 自動保存最近使用的 20 個顏色，點擊即可快速載入
- **調色板生成** - 自動生成單色調色板和互補色調色板
- **深色主題** - 精美的深色介面設計

## 安裝步驟

### 1. 生成圖標

在安裝擴充套件之前，你需要先生成圖標文件：

1. 打開 `icons/generate-icons.html` 文件（使用瀏覽器開啟）
2. 點擊每個「下載」按鈕，下載三個尺寸的圖標
3. 將下載的圖標放到 `icons` 資料夾中
4. 確保檔名正確：`icon16.png`、`icon48.png`、`icon128.png`

### 2. 載入擴充套件到 Chrome

1. 開啟 Chrome 瀏覽器
2. 在網址列輸入 `chrome://extensions/`
3. 開啟右上角的「開發人員模式」
4. 點擊「載入未封裝項目」
5. 選擇這個專案資料夾
6. 擴充套件圖標會出現在瀏覽器工具列上

## 使用方法

### 基本使用

1. **開啟擴充套件** - 點擊工具列上的 Color Picker 圖標
2. **選擇顏色** - 使用顏色選擇器調整色相、飽和度和亮度
3. **複製顏色** - 點擊任何格式旁的複製按鈕

### 網頁取色

1. 點擊「PICK COLOR」按鈕
2. 將滑鼠移到網頁上的任何位置
3. 會顯示放大鏡和顏色資訊
4. 點擊滑鼠即可選取該顏色
5. 按 ESC 可取消取色

### RGB 直接輸入

在 RGB 輸入框中直接輸入數值（0-255），顏色會即時更新。

### 調色板生成

- **Monochromatic（單色調）** - 生成基於當前顏色的不同亮度變化
- **Complementary（互補色）** - 生成互補色、分裂互補色和三色組
- 點擊調色板中的任何顏色即可複製其 HEX 值

### 顏色歷史記錄

- 自動保存從網頁取色和調色板點擊的顏色
- 最多保存 20 個顏色，超過後自動移除最舊的記錄
- 智能去重：重複的顏色會移到最前面而不是重複顯示
- 點擊歷史記錄中的任何顏色即可載入到選擇器中
- 歷史記錄會持久保存，關閉瀏覽器後再次開啟仍然保留
- 多個擴充套件視窗會自動同步歷史記錄

## 技術細節

- **Manifest Version**: 3
- **Permissions**: activeTab, storage
- **支援的瀏覽器**: Chrome, Edge, Brave 等基於 Chromium 的瀏覽器

## 顏色格式說明

- **HEX** - 十六進位制格式，如 `#3B82F6`
- **RGB** - 紅綠藍格式，如 `rgb(59, 130, 246)`
- **HSL** - 色相飽和度亮度格式，如 `hsl(217°, 70%, 50%)`

## 開發

如果你想修改或擴展這個擴充套件：

1. 修改 `popup.html`、`popup.css`、`popup.js` 來調整彈出視窗
2. 修改 `content.js` 來調整網頁取色功能
3. 修改 `manifest.json` 來添加新的權限或功能
4. 在 Chrome 擴充套件頁面點擊「重新載入」來更新

## 常見問題

**Q: 為什麼取色功能不工作？**
A: 某些網頁（如 Chrome 內建頁面）不允許擴充套件注入腳本。請在一般網頁上使用。

**Q: 顏色歷史記錄保存在哪裡？**
A: 歷史記錄使用 Chrome Storage API 保存在本地，最多保存 20 個顏色。資料會持久保存，即使關閉瀏覽器也不會遺失。

**Q: 為什麼移動滑桿不會添加到歷史記錄？**
A: 為了避免歷史記錄過於雜亂，只有從網頁取色和點擊調色板顏色時才會添加到歷史。手動調整滑桿或輸入 RGB 值不會觸發歷史記錄。

**Q: 支援其他瀏覽器嗎？**
A: 理論上支援所有基於 Chromium 的瀏覽器（Edge, Brave 等），但 Firefox 需要修改 manifest.json。
